Home weather station on [ESP32](https://en.wikipedia.org/wiki/ESP32).

---

### Features:

- environment readings:
  - temperature
  - humidity
  - pressure
  - co2
 - saving readings to database
 - on-board display to show current readings
 - browser web app to show graphs
